Заметки по верстке (что сделано и зачем)

1) Общая цель:
- Создать компонент QR-кода в виде карточки, соответствующий макету из папки `./design`.
- Обеспечить корректную семантическую разметку, адаптивность и базовую типографику.

2) Файлы, которые были добавлены/изменены:
- `index.html` — заменена базовая заглушка контента на семантическую структуру карточки:
  - `<main class="page">` — центрирует контент по странице;
  - `<section class="card">` — блок карточки с QR-изображением, заголовком и описанием;
  - подключён шрифт `Outfit` через Google Fonts;
  - добавлена ссылка на внешний CSS `styles.css`.

- `styles.css` — новый файл с базовыми и адаптивными стилями:
  - определены CSS-переменные цветов на основе `style-guide.md` (фон, основные цвета текста);
  - сброс box-sizing и базовые стили для `body` (центровка, фон, отступы);
  - стили карточки: фон белый, скругления, тень, фиксированная ширина для визуального соответствия макету;
  - стиль для изображения QR: максимальная ширина, скругление, центрирование;
  - типографика: размеры заголовка и абзаца согласно гайду (задан базовый `font-size` для параграфов 15px);
  - медиазапрос для улучшения отображения на планшете/десктопе (увеличены отступы/ширина).

- `CHECKLIST.md` — файл чек-листа (русский) создан для отслеживания прогресса по задачам.
- `SUMMARY.md` — краткая сводка изменений и инструкции по проверке (создан для документации).

3) Почему я сделал именно так (обоснование решений):
- Семантика (`main`, `section`, `h1`, `p`) помогает доступности и SEO, облегчает навигацию по DOM.
- Подключение Google Font (`Outfit`) соответствует `style-guide.md` — это улучшает визуальное соответствие макету.
- CSS-переменные упрощают поддержку цвета и быстрые правки по всей теме.
- Центровка карточки и ограничение ширины делают компонент ближе к макету и проще для точной подгонки.
- Тень и скругления добавлены для визуального соответствия стилю карточек в макете.
- Мобильный-first подход (стили изначально ориентированы на мобильный макет) — проще поддерживать адаптивность.
